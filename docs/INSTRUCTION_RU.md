# Инструкция пользователя (живая)

Этот документ обновляется на каждой итерации.

## Последовательность действий (актуально)

1. Войти как главный админ.
2. Открыть раздел **Администрирование** в левом меню.
3. Нажать **Быстрый мастер**.
4. Ввести email подрядчика, email заказчика и пароль.
5. Нажать **Создать демо-процесс**.
6. Открыть систему в двух окнах браузера:
   - обычное окно: подрядчик
   - инкогнито: заказчик
7. Проверить процесс:
   - MDR
   - Document
   - Revision
   - Comment
   - Response
8. Проверить раздел **Уведомления**.

## Ссылка на публичную инструкцию

- В веб-приложении: раздел **Инструкция** → кнопка "Открыть публичную инструкцию".
- Прямой путь: `/instruction.html` (на frontend-домене).
