# Инструкция для пользователей системы TDO

Документ предназначен для конечных пользователей проекта.

## Шаг 1. Карточка проекта

1. Главный админ открывает раздел **Проекты**.
2. Нажимает **Создать проект**.
3. Заполняет:
   - код проекта,
   - название,
   - описание,
   - руководителя ТДО подрядчика (опционально).
4. Сохраняет карточку проекта.

## Шаг 2. Участники проекта

1. Открыть карточку проекта → вкладка **Участники проекта**.
2. Главный админ назначает руководителя ТДО подрядчика (если не назначили при создании).
3. Руководитель ТДО подрядчика добавляет участников подрядчика.
4. Главный админ добавляет пользователей заказчика.
5. При необходимости участника можно удалить кнопкой **Удалить**.

## Шаг 3. Справочники проекта

1. Открыть карточку проекта → вкладка **Справочники проекта**.
2. Базовые справочники автоматически добавляются при создании проекта.
3. Главный админ добавляет/редактирует:
   - дисциплины,
   - типы документов,
   - категории и шаблоны идентификаторов,
   - и другие значения.

## Шаг 4. Работа с документацией

1. Подрядчик формирует MDR по коду проекта.
2. Подрядчик создает документ.
3. Подрядчик создает ревизию.
4. Подрядчик загружает PDF в ревизию (кнопка **PDF**).
5. Заказчик добавляет комментарии.
6. Подрядчик отвечает на комментарии до согласования.

## Шаг 5. Удаление карточки проекта

Главный админ может удалить карточку проекта, если по проекту еще нет MDR-записей.
